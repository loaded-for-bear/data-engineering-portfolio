# 03. CSV ETL（欠損値・分類）

## 概要

ECサイトのアクセスログを読み込み、欠損値の補完・カテゴリ分類・ユーザー別/時間帯別の集計レポートを作成するETLパイプライン。

## 使用技術

- pandas: `fillna`, `transform`, `groupby`, `agg`
- 欠損値の中央値補完（アクション種別ごと）
- 時間帯抽出・カテゴリ分類

## 入力データ

- `data/access_log.csv` — Webアクセスログ（一部に欠損・不正値を含む）

## 出力

- `user_summary.csv` — ユーザー別メトリクス（閲覧数、滞在時間、購入フラグ）
- `hourly_report.csv` — 時間帯別レポート（リクエスト数、ユニークユーザー数、平均滞在時間）

## 実行方法

```bash
python etl_pipeline.py
```

## 処理内容

1. **Extract**: CSV読み込み、不正レコードのフィルタリング
2. **Transform**: 欠損値をアクション種別の中央値で補完、ページカテゴリの分類、時間帯の抽出
3. **Load**: ユーザー別・時間帯別の集計CSVを出力
